# Omni Food CSS Project



## Overview

**Omni Food** is a modern, responsive landing page for a food delivery service built with **HTML5**, **CSS3**, and **vanilla JavaScript**.  
This project is part of the **Udemy CSS Course by Jonas Schmedtmann**, designed to demonstrate clean and modular CSS architecture, mobile-first design, and interactive UI features.

---

## Features

- Fully **responsive design** for desktop, tablet, and mobile
- **Sticky header** with smooth scroll navigation
- **Hero section** with CTA buttons
- **Meal section** showcasing dishes with attributes and diet labels
- **Testimonial section** with image gallery
- **Pricing section** with multiple plans and feature highlights
- **Call-to-Action (CTA) form** with input validation
- **Lazy-loading image gallery** for optimized performance
- **Mobile navigation toggle** for smaller screens
- Modular CSS structure (`style.css`, `general.css`, `meals-list.css`, etc.)
- **Smooth scroll polyfill** for cross-browser support
- Integrated **Ionicons** for icons
